# CourtManager Pro

## SaaS de Utilería Profesional para Baloncesto

**Ramón del Pozo Rott** · Creador & Superadmin  
**Carlos Rodriguez Kobe** · Utilero RMB · Validación operativa

🌐 courtmanagerpro.vercel.app · Junio 2026


# El problema

Los clubes de élite pierden **€15.000 – €30.000 al año** en:

- Material extraviado en aeropuertos y viajes

- Zapatillas y equipación duplicada o sin trazabilidad

- Errores de tallaje antes de partidos críticos

- Retrasos de lavandería y roturas de stock

**La utilería es logística de alto valor — y hoy se gestiona con Excel y WhatsApp.**


# Océano azul

| Existe mucho… | No existe (hasta ahora)… |
| - | - |
| Apps de estadísticas | SaaS solo de **utilería** |
| Scouting y fichajes | **26+ tallas** por jugador |
| ERP genéricos | **QR + inventario** deportivo |
|  | Alertas cumpleaños, viajes, botiquines |


> *CourtManager Pro ocupa un nicho desatendido con disposición a pagar muy alta.*


# La solución

**CourtManager Pro** — plataforma integral:

- 📋 Tabla de tallas (equipación oficial + entrenamiento + accesorios)

- 📦 Inventario con códigos QR / barras

- ✈️ Viajes y packing list

- 🧺 Lavandería · 🏥 Material médico y botiquines

- 🔔 Alertas (stock, caducidades, **cumpleaños plantilla**)

- 📊 Informes de equipación exportables

**Demo en producción:** Real Madrid Baloncesto 2025/26

# Producto — MVP en vivo

✅ **35 rutas** desplegadas en Vercel  
✅ **0 errores** de compilación  
✅ **PWA instalable** (App desde navegador)  
✅ **17 jugadores** + 8 staff + inventario demo  
✅ **15 referencias médicas** + 10 solicitudes workflow  
✅ Auditoría técnica completa (`docs/auditoria/`)

Stack: **Next.js 15 · React 19 · Supabase · Vercel**
