# Clubes Liga Endesa (ACB) — Contactos y envío de propuesta

**Temporada de referencia:** 2025-2026  
**Uso:** Lista genérica para outreach de CourtManager Pro  
**Remitente:** Ramón del Pozo Rott · info@ramondelpozorott.es

> **Importante:** Los emails directos de utilería suelen ser internos. La vía más fiable es el **formulario de contacto** de la web oficial o **LinkedIn** (Director deportivo / Responsable de operaciones). Verifica siempre en la web del club antes de enviar.

---

## Cómo usar esta lista

1. Personaliza el email con el nombre del club ([ver plantilla](../acb-outreach/README.md))
2. Adjunta o enlaza: dossier PDF, presupuesto, one-pager
3. Incluye enlace a presentación (ES / CA / EN / FCB)
4. Incluye demo: https://courtmanagerpro.vercel.app
5. Para **FC Barcelona** → catalán obligatorio ([dossier FCB](./fc-barcelona/DOSSIER-FCB-CA.md))

## Perfiles a buscar en LinkedIn

Para cada club, buscar y contactar (mensaje corto + enlace demo):

| Perfil | Por qué |
|--------|---------|
| **Director deportivo** | Decisor presupuesto operativo |
| **Jefe de utilería / Equipment manager** | Usuario principal |
| **Director de operaciones** | Logística y viajes |
| **Responsable de cantera** | Plan Starter / upsell |
| **CEO / Gerente** | Clubes medianos (Manresa, Girona, etc.) |

**Mensaje LinkedIn (ES):**

> Hola [Nombre], he desarrollado CourtManager Pro, SaaS de utilería para clubes ACB (inventario QR, tallas, viajes). Demo: courtmanagerpro.vercel.app — ¿Te interesa una demo de 30 min para [Club]?
